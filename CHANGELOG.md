# Changelog

All notable changes to this project will be documented in this file.

## 1.0.0 (Unreleased)

### Features

* Initial release
* Support for IAM users creation with comprehensive configuration options
* Support for IAM groups creation
* Support for group memberships (assigning users to groups)
* Support for user role assignments at domain, project, or enterprise project level
* Support for group role assignments at domain, project, or enterprise project level
* Support for user access types (default, programmatic, console)
* Support for login protection (SMS, email, MFA)
* Support for external identity providers (SSO)
* Comprehensive tagging support for all resources
